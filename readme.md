# real time geo apis
a [talk](http://jden.us/speaking) by jden
delivered 2014-09-09 at JSgeo in Portland, OR, USA.

## running the demo
requires: node.js

1. clone the repo
2. cd demo
3. npm install
4. npm start

There are multiple steps in the demo, so you can also do `npm run 1` ... `npm run 4`
